Commanding robot motion based on visual input

Logic: 
    1. If tag 0 is detected right turn 90 degrees
    2. If tag 1 is detected left turn 90 degrees
    3. After completion continue moving forward until the next tag is detected
    4. Repeat this for 5 detected tags
